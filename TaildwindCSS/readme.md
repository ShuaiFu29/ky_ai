# TaildwindCSS 原子CSS

- 非常的好用
- 几乎不用写CSS
- AI 生成代码 CSS 用的都是taildwindCSS
- 配置流程
    tailwindcss @tailwindcss/vite
- 有各种内置的CSS 类名  分门别类
- 1rem = 4个单位

- 文字行数限制
    -webkit-line-clamp: 2;  不能独自生效
    -webkit 浏览器内核   Chrome + Safari
    -mozi 火狐浏览器内核代号 
    实验阶段的属性  新的
    display: -webkit-box;
    -webkit-box-orient: vertical;
    overflow: hidden;
